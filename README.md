# catctl
A command line utility to control amateur radios.

## About/Why
I'm writing this utility because I want a fast, scriptable way to interact with
my radio.  There are a lot of very good GUI apps for amateur radios, but
a CLI is the right tool for some jobs.  Making one is a good way to practice
my software development and make something useful.

## What radios does it support?
As of right now, it supports the FT450D.  I'm trying to write it in such a
manner that it will support other radios by adding config files.

## What is the status of this project?
It's currently not feature-complete, but I hope for it to be soon.

## Are there any releases?
Not yet.  Once I get the basic features working there will be.

## How do I install it?
If you have the required tooling installed `make build` ought to do the job for
 you.
